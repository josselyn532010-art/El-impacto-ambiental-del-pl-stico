Este repositorio explora de manera detallada el impacto ambiental del plástico, ofreciendo recursos, estudios y análisis que ayudan a comprender sus efectos y posibles soluciones.

En este repositorio encontrarás información sobre cómo el plástico afecta los ecosistemas marinos, las posibles alternativas sostenibles y las iniciativas globales que están surgiendo para reducir el uso de plástico
